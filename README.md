# Nivra

AI-powered voice notes & transcription app for iOS.

Record lectures, meetings, or any audio — get instant AI-powered transcriptions, summaries, quizzes, flashcards, and key points.

## Features

- 🎙️ **Record & Transcribe** - 150+ languages, powered by Groq Whisper
- 📺 **YouTube Import** - Get transcripts from any YouTube video
- ✨ **AI Summaries** - 6 styles: Concise, Detailed, Bullet Points, Exam Prep, Cornell Notes, Mind Map
- 🧠 **Quizzes & Flashcards** - Auto-generated from your content
- 📂 **Smart Folders** - Organize by subject
- 📥 **Export** - TXT, SRT, VTT formats
